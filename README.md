This project contains the codes (0\_\*, 1\_\* and 2\_\*) to compute the drought mass and optimize the network, associated with synthetic time series of solar radiation and wind, with more (or less) dunkleflaute events.
- 0_Droughtmass_and_Network_obj_value.py: the codes to compute drought mass and optimize network, in a .py format (essentially contains the same information as 1_Modified_complete_codes.ipynb)
- 1_Modified_complete_codes.ipynb: the codes to compute drought mass and optimize network, in a .ipynb format. <<-- this code was run to create all the files of the type synthetic03xless.csv
- 2_Plot_Results_droughts_vs_cost_each_solar_ratio.ipynb: the codes to plot the results of the optimization: drought mass VS objective value, for different solar ratio. <<-- this code was used to create all teh figures, files of the type Objective_vs_DroughtMass_SolarRatio_Dunkleflaute03xless.pdf
